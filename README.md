### 一、前言

**本项目基于`Spring Boot`+`JWT`+`Vue`完成了一个在线笔记发布平台**

###### 功能

1. 登录注册
2. 可在线发布笔记
3. 可查看，点赞，评论笔记
4. 可查看自己发布的笔记，以及收藏的笔记
5. 笔记分组功能

###### 开发环境：

1. JDK 1.8
2. IDEA 2022.1
3. MySQL 8
4. Node.js 14.10.0

###### 项目使用技术栈：

前端：`Vue` + `Axios` + `Element-ui` + `mavon-editor`
后端：`Spring Boot` 、 `MyBatis`
.......
